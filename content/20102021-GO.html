A nova queridinha do mercado, conheça a linguagem de programação GO.

20/10/2021

Muitas vezes conhecida por “golang do Google”, a linguagem de programação GO é uma linguagem que teve sua popularidade aumentada demais nos últimos anos. Mesmo com linguagens antigas que ainda dominam o mundo de programação, como Java e C, existem novas linguagens surgindo, e se adaptando melhor aos novos recursos disponíveis, o que é o caso da GO.

O aumento da procura e uso da linguagem GO é uma consequência de sua linguagem simples e leve, com código aberto, muitas vezes adequado às arquiteturas de microsserviços. Como grandes exemplos que foram construídos com o GO, podemos citar Docker e Kubernetes.

A linguagem de programação Go é uma linguagem que também cresce no ambiente de ciência de dados, abrangendo especialidades como inteligência artificial. Quando comparada com outras linguagens, como Python e R, o GO apresenta diversos pontos fortes em relação às suas concorrentes, podemos citar o tratamento de erros e o debug facilitado como exemplos.

Se destacando em quase todos os aspectos, a linguagem de programação GO veio destinada a resolver problemas com linguagens e ferramentas existentes, aproveitando sempre a as arquiteturas de hardware modernas.

Bom, só de observar um pouco sobre a linguagem GO podemos entender o motivo dela estar tão em alta no mercado! Gostaria de aprender mais sobre ela? Dá uma conferida nesse livro que recomendamos: A Linguagem de Programação Go, de 14 de março de 2017, por Alan A. A. Donovan (Autor), Brian W. Kernighan (Autor)., pela Amazon.
